# Meow Paint App

https://editor.p5js.org/jeanchen0613/full/GXXi5wx6H

An interactive web painting application built with p5.js, featuring custom spray brushes.

## Controls
- **Mouse Press** : Paint with the current spray brush
- **`r`** : Change to a random brush color
- **`c`** : Clear the drawing layer (Keeps the background intact)
- **`s`** : Save the current canvas as a `.png` file
